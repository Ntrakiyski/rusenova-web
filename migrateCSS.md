## Why This Works

- **`@import "tailwindcss"`** in a separate file (`styles/tailwind.css`) gives Tailwind its own isolated space to expand its giant generated CSS
- **Custom styles** in `styles/theme.css` are kept separate, so PostCSS/Turbopack doesn't choke on the massive Tailwind output
- **`app/globals.css`** just re-exports both, which Next.js imports cleanly in `layout.tsx`

This matches the **recommended Tailwind v4 + Next.js 16 + Turbopack** pattern.
